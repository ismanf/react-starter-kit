# react-starter-kit
React-redux boilerplate app with latest trends for beginners.... happy coding!!!

Clone or download
```
git clone https://github.com/ismayilmalik/react-starter-kit.git

```

Install dependencies with ***npm***:

```
npm install
```

with ***yarn*** :

```
yarn install
```

Build:
```
npm run build
```

Development server:
```
npm run start:dev
```
